[ Edit ](https://docs.frappe.io/wiki/spaces/24hrpr6es9/page/0rl82b39ti)

ERPNext comes with 2 predefined configuration for Currency Exchange Rate providers: [Frankfurter.app](https://www.frankfurter.app/) and [Exchangerate.host](http://exchangerate.host/)

# [Frankfurter.app](http://frankfurter.app/)

This is free exchange rate provider. But, it provides exchange rates for a limited set of currencies.
