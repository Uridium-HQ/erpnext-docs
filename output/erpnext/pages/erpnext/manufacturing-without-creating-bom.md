# Manufacturing without creating BOM

[ Edit ](https://docs.frappe.io/wiki/spaces/24hrpr6es9/page/0sp97mgmft)

Open in ChatGPT  Ask ChatGPT about this page Open in Claude  Ask Claude about this page

**It is recommended to use BOM for Manufacturing for consistency and to avoid repetitive work.**  
However, if you wish to perform it manually, the workaround is to create Stock Entries:1) Create a **Stock Entry** with **Purpose** as **Material Transfer to Manufacture.**![](../../assets/c41517a401f2.png) 2) Create a **Stock Entry** with **Purpose** as **Manufacture, the Item which is being Manufactured will have a Target Warehouse and other Items(Raw Materials) will have a Source Warehouse.**  
![](../../assets/e46098484f12.png)  
_﻿Blue Dot: Item is being received._ _Orange Dot: Item is being issued._
