# Blocks

[ Edit ](https://docs.frappe.io/wiki/spaces/r3uvq1ch61/page/12n4n37fge)

Open in ChatGPT  Ask ChatGPT about this page Open in Claude  Ask Claude about this page

Blocks as the name suggest are the building blocks of the workspace. You can add different types of blocks, edit them, and arrange them as per your need to build a fully customizable workspace.

There are currently 9 blocks. We will keep adding new blocks in the coming future.

Let's go through each of them:

![Blocks](../../../../../../../assets/ed913978a7d7.png)

###Heading A heading block is used as a header for any section or paragraph. ![Section Header](../../../../../../../assets/39db95eb801f.png) ![Paragraph Header](../../../../../../../assets/7fc929a7cb69.png)

###Text Text block is used as a paragraph or description for any section. ![Paragraph Block](../../../../../../../assets/7b3b7f216994.png)

Both **Heading** and **Text** has an inline toolbar. Currently, we have limited tools like header size from H1 to H6, bold, italic and links. ![Inline Toolbar](../../../../../../../assets/b283dc7f8fb0.png)

###Card A card contains several links that can be used for quick navigation. The link item can be a DocType, Report or Page. ![Card Block](../../../../../../../assets/dc514372d6f5.png)

###Chart You can add any **Dashboard Chart** on the workspace using this block. ![Chart Block](../../../../../../../assets/8e06e40ec54b.png) ###Shortcut Shortcut block is used to quickly navigate to any DocType, Report or Page. ![Shortcut Block](../../../../../../../assets/6c3f28fe6293.png) Shortcuts can be customized too. You can select the view in which it should open, and can add filters that will be applied when you open the selected view. You can see the number of records based on the filters applied on the shortcut as a pill. ![Shortcut Features](../../../../../../../assets/7b287eed4a76.png) ###Spacer It is used to add some space between two blocks. ![Spacer Block](../../../../../../../assets/bbb64c300cae.png) It can also be used to position other blocks by adding white space before or after the block. ![Spacer Separator](../../../../../../../assets/77fda3e5822b.png) ![Spacer Separator Output](../../../../../../../assets/b3de1faf06fc.png) ###Onboarding The onboarding block is used to guide first-time users when they land on your page. It has steps to follow to learn a specific module. For each step, there is some description on the right section or there can be a link to a video. ![Onboarding Block](../../../../../../../assets/1199d68dbd90.png) ###Quick List In the quick list block, we can see recently updated records of selected doctype. We can also add some filters. While in the read-only mode we have the option to refresh the list, update the applied filters, create a new record and open doctype's list view. ![Quick List Block](../../../../../../../assets/8ac7769b5596.png) ###Number Card You can add any existing **Number Card** on the workspace using this block. ![Number Card Block](../../../../../../../assets/5b3ef087a832.png)
